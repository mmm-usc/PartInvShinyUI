Some notes on notation and examples of notation use

Selection accuracy indices (SAI) will be referred to by a two letter acronym e.g., \(\text{SE}\), and ‘overall’ accuracy indices will be denoted with a tilde over the acronym, (e.g., \(\tilde{\text{SE}}\), \(\tilde{\text{SR}}\)). Unless otherwise indicated, we will assume that the full item set is used and that partial factorial invariance (PFI) holds.

If multiple operations are performed and multiple properties need to be indicated the outermost superscript indicates the current operation, and innermost superscript refers to constant properties. In other words, the superscript of \(h\) or \(\Delta h\) indicates the current comparison (e.g., full vs. modified item set, strict vs. partial invariance, reference vs. focal groups). The superscript of an SAI refers to properties that are constant in in both groups being compared, and the subscript (if any) indicates that \(h\) was computed on this SAI to compare the conditions indicated in the subscript.

Indicating the set of items

  • Let \(I\) denote a set of items in a p-element scale, \(I = \{i_1, i_2,\ldots,i_p\}\). Then, we can denote the modified set of items after removing the k-th item as \(I^{|k}=\{i_1, i_2, ..., i_{k-1},i_{k+1}, ..., i_p\}\). Using this notation, we can refer to a selection accuracy index (SAI) computed using the full set of items as ‘SAI’ (e.g., SE), and the SAI computed using the modified set of items excluding item k as \(\text{SAI}^{|k}\) (e.g., \(\text{SE}^{|2}\) if the modified item set excludes the second item).

Operations comparing the full vs. modified item sets

  • \(h(\text{SAI},\text{SAI}^{|k})\) is denoted as \(h^{|k}(\text{SAI})\), and the resulting h values are indicated as \(\text{SAI}_{|k}\).

  • \(\Delta h(\text{SAI},\text{SAI}^{|k})\) is denoted as \(\Delta h^{|k}(\text{SAI})\), and the resulting \(\Delta h\) values as \(\Delta h^{|k}\text{SAI}\).

  • Note: \(\Delta h\) is computed only for the comparison between the full and modified item sets (not for group or invariance condition comparisons).

  • e.g., \(\Delta h^{\text{|2}}(\text{SAI}_{\text{r-Ef}})\) for quantifying the impact of dropping the second item on the comparison between SAI for the r and Ef groups under PFI: \(\Delta h^{|2}\text{SAI}_{\text{r-Ef}}\).

  • e.g., for the reference group, \(h\) comparing SAI under SFI vs. PFI followed by \(\Delta h\) comparing these h values computed for the full item set and for the modified item set excluding the 4th item: \(\Delta h^{|4}\text{SAI}^{r}_{\text{str, par}}\).

  • e.g. \(h^{|3}(\tilde{\text{SAI}})\) for comparing the overall SAI under PFI: \(\tilde{\text{SAI}}_{|3}\).

h comparing the SAI of groups

  • Reference group (r) can be compared with the focal group (f) or Efocal (Ef).

  • \(h^{\text{r-Ef}}(\text{SAI}) = \text{SAI}_{\text{r-Ef}}\) for h comparing the SAI of r and Ef groups under PFI, for the full item set.

  • \(h^{\text{r-Ef}}(\text{SAI}^{|2})=\text{SAI}_{\text{r-Ef}}^{|2}\) for h comparing the SAI of r and Ef groups for modified item set excluding the second item.

  • \(h^{\text{r-f}}(\text{SAI}^{\text{str}}) = \text{SAI}_{\text{r-f}}^{\text{str}}\) for h comparing the SAI of r and f groups under SFI, for the full item set.

  • \(h^{\text{r-f}}(\text{SAI}^{|2;\text{str}}) = \text{SAI}_{\text{r-f}}^{|2; \text{str}}\) for h comparing the SAI of r and f groups under SFI, for the modified item set excluding the second item.

h comparing the SAI under SFI vs. PFI

  • \(h^{\text{str, par}}(\text{SAI}^r)=\text{SE}_{\text{str, par}}^r\) for h comparing the SAI under SFI vs. PFI (for the reference group).
  • \(h^{\text{str, par}}(\text{SAI}^{|2; r})=\text{SE}_{\text{str, par}}^{|2; r}\) for h comparing the SAI of r and Ef groups for modified item set excluding the second item.

h_overall_par

\(h^{|k}(\tilde{\text{SAI}})\): Cohen’s h effect size for the improvement/drop in overall SAI under PFI when the k-th item is removed. ‘Overall’ refers to the averaging of accuracy indices across focal and reference groups by group proportions to arrive at a weighted average for each SAI (regularly computed as \(\text{SR}=\frac{\text{TP}}{\text{TP}+\text{FP}}\), \(\text{SE}=\frac{\text{TP}}{\text{TP}+\text{FN}}\), \(\text{SR}=\frac{\text{TN}}{\text{TN}+\text{FP}}\)). Letting \(p_r\) denote the weight for the reference group,

\[\tilde{\text{SR}} = \frac{\text{TP}_{r}\times p_r + \text{TP}_{f}\times (1-p_r)}{\text{TP}_{r}\times p_r + \text{TP}_{f}\times (1-p_r) + \text{FP}_{r}\times p_r + \text{FP}_{f}\times (1-p_r)}\] \[\tilde{\text{SE}} = \frac{\text{TP}_{r}\times p_r + \text{TP}_{f}\times (1-p_r)}{\text{TP}_{r}\times p_r + \text{TP}_{f}\times (1-p_r) + \text{FN}_{r}\times p_r + \text{FN}_{f}\times (1-p_r)}\] \[\tilde{\text{SP}} = \frac{\text{TN}_{r}\times p_r + \text{TN}_{f}\times (1-p_r)}{\text{TN}_{r}\times p_r + \text{TN}_{f}\times (1-p_r) + \text{FP}_{r}\times p_r + \text{FP}_{f}\times (1-p_r)}\]

Then, the change in overall SAI after dropping the k-th item can be computed as \[h^{|k}\tilde{\text{SAI}} = 2\arcsin\left(\sqrt{\tilde{\text{SAI}}}\right)-2\arcsin\left(\sqrt{\tilde{\text{SAI}}^{|k}}\right)\]

A negative h_overall_sai_par indicates an improvement (as \(h^{|k}\tilde{\text{SAI}}<0 \iff \tilde{\text{SAI}}<\tilde{\text{SAI}}^{|k}\)) whereas a positive h_overall_sai_par indicates a deterioration in performance.

For the 4-item CES-D scale, the negative \(h^{|1}(\tilde{\text{SR}})\), \(h^{|1}(\tilde{\text{SE}})\), and \(h^{|1}(\tilde{\text{SP}})\) values indicate that removing the first item would lead to an improvement in SR, SE, and SP. On the other hand, the positive \(h\) values for modified item sets excluding items 2, 3, or 4 suggest that SR, SE, and SP decreases if any of these items are dropped.

delta_h

\(\Delta h\) is computed using the following formula to compare the effect size of the change in an SAI when the k=th item is dropped: \[\Delta h^{|k}(\text{SAI})=\text{sign}\left(| h^{(.)}({\text{SAI})}|-|h^{(.)}(\text{SAI}^{|k})|\right)\left||h^{(.)}(\text{SAI}^{|k})|-|h^{(.)}(\text{SAI})|\right|\] ‘(.)’ in the superscript of h is a placeholder for the comparison performed in the previous step.

delta_h contains two elements: delta_h$h_R_vs_Ef.par and delta_h$h_str_vs_par, with the latter containing two tables (one for reference and the second for focal group comparisons).

delta_h$h_R_vs_Ef.par
